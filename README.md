# Todo List

一个轻量级的网页应用，用于记录待办事项（Todo 事件）。

## 项目简介

- **前端部分**  
  因为前端基础较为薄弱，参考了 B 站 UP 主 **“三分钟实验室”** 的教程完成。

- **后端部分**  
  借助自己的后端基础，使用 **Spring Boot** 实现了一个简单的增删改查（CRUD）功能。

## 收获与感悟

通过这个项目，我体会到了开发一个完整网页的魅力，受益匪浅~🌟

### 技术栈

- **前端**: Vue3  
- **后端**: Spring Boot + MySQL + Mybatis
- **其他**: RESTful API 设计
